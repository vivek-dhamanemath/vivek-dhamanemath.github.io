Overview

ArtistryX is a full-stack web application built to streamline artist and film management workflows. With a modern UI, secure authentication, and powerful backend integration, ArtistryX delivers a professional-grade solution for managing artistic data and subscriptions.

Tech Stack

Next.js React TailwindCSS Spring Boot PostgreSQL Stripe

Key Features

  • Secure login with JWT & Google OAuth2
  • Artist & Film CRUD management
  • Export artist data in multiple formats
  • Stripe-powered subscription management
  • Responsive UI with modern animations

Deployment

Frontend: Vercel

Backend: Render

Database: Cloud PostgreSQL
